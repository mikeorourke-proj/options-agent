# Options Positioning Intelligence Agent

JonesTrading options positioning dashboard. Type any ticker to see how options activity is influencing share price.

## Setup

1. Deploy to Netlify (connect this repo)
2. Add environment variables in Netlify:
   - `FLASHALPHA_API_KEY` — from [flashalpha.com](https://flashalpha.com/pricing)
   - `POLYGON_API_KEY` — from [polygon.io](https://polygon.io/pricing?product=options)
3. Deploy triggers automatically on push
